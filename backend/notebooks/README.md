# Telecom Customer Churn Prediction System

An end-to-end ML project to predict telecom customer churn using machine learning, served through a FastAPI backend and a React frontend.

## Tech Stack
- Python
- FastAPI
- Scikit-learn
- XGBoost
- React
- Vite

## Project Goals
- Predict customer churn
- Build a reusable ML pipeline
- Serve predictions through API
- Show predictions in a frontend dashboard
